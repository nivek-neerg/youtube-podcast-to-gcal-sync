PodSync

Automatically schedule new YouTube podcast episodes into your Google Calendar — around your existing events, within hours that work for you.

Step 01 API Keys

Two free keys from Google are needed. Open Google Cloud Console →
Enable YouTube Data API v3 and Google Calendar API. Create an API Key for YouTube and an OAuth 2.0 Client ID for Calendar.

Step 02 YouTube Channels

Paste a channel URL and click Look up to load its name and picture. Use the toggle on each channel to pause or resume it without removing it.

Step 03 Your Schedule

Choose which days and time window episodes can be booked into.
